# Random Persian Text Generator
## Install requirements
``
pip install -r requirements.txt
``
# Usage
``
usage: main.py [-h] [-l LENGTH]
``

optional arguments:
  -h, --help            show this help message and exit
  -l LENGTH, --length LENGTH
                        text length

## Sample
``
python main.py
``

``
python main.py -l 30
``

``
python main.py --length 40
``
